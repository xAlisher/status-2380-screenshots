4 · Bugs & Issues Found

🔐
Seed phrase words exposed in accessibility tree before "Reveal" tap
All 12 words readable via uiautomator dump without any user interaction. Any app with accessibility permission can silently extract the seed phrase. → #20931
🐛
Bio field accepts input beyond 240-character limit
No hard cap — field shows 243/240 with red border and blocks save, but does not stop typing. User must manually trim. → #20932
⚠️
No confirmation when navigating away with unsaved changes
Back button exits Edit Profile silently. Changes are retained in memory but user is not informed they were not saved. → #20933
🐛
"Drag communities here" cross-section drag non-functional on Android
The label "Drag communities here to display in showcase" creates a false affordance. Dragging from Hidden → In showcase has no effect. Workaround: use the visibility dropdown. → #20934
⚠️
"Enable push notifications" modal cannot be dismissed by tapping outside
Standard Android bottom sheet dismiss (tap overlay) is ignored. Must use "Maybe later" or "Continue" buttons. → #20935
🐛
"Changes detected" bar clips text at left screen edge
The dirty-state bottom bar text is cut off on the left side. → #20927
